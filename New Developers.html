To become a developer you must follow these steps

  1. Create a GitHub account
  2. Learn to code (if you know how skip this step)
  3. Make an app (name it "app name".github.io)
  4. Upload it to GitHub
  5. Go to settings in GitHub of the project
  6. Scroll down to GitHub pages
  7. Click the drop down that says "NONE"
  8. Then select master page
  9. Copy the link and paste it into an email to me
  10. Then the app will be uploaded some time that week
